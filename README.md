# BOJ
[백준](https://www.acmicpc.net/) 문제 풀이를 위한 레포지토리

#### 템플릿 코드 생성

```bash
$ ./gen <백준 문제 번호>
```

#### 컴파일 및 실행
```bash
$ ./run <백준 문제 번호>
```

#### 컴파일 및 실행(input.txt 값 입력)
```bash
$ ./run -f <백준 문제 번호>
```

#### 소스 코드 포멧팅(clang-format 필요)
```bash
$ ./format
```
